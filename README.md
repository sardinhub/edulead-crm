# EduLead CRM - Marketing Follow-Up System

Sistem manajemen konversi siswa dari pendaftaran → pembayaran.

## Cara Menjalankan (Local Development)

Pastikan Anda memiliki [Node.js](https://nodejs.org/) terinstall.

1. Buka terminal di folder project ini.
2. Install dependensi dengan perintah:
   ```bash
   npm install
   ```
3. Jalankan development server:
   ```bash
   npm run dev
   ```
4. Buka URL yang muncul di terminal (biasanya `http://localhost:5173`) di browser Anda.

## Teknologi (UI/UX Phase)
- React
- Vite
- Tailwind CSS (Light Mode Base)
- Framer Motion (Animations)
- Lucide React (Icons)
- Zustand (For state management integration later)

Sistem telah dilengkapi dengan mockup pipeline `Pendaftaran -> DP Pangkal -> Pangkal Lunas` dengan gaya tampilan Kanban Board yang interaktif sesuai App Plan.
